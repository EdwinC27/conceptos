Programación

Qué es?

La programación es el proceso utilizado para idear y ordenar las acciones necesarias para realizar un proyecto, preparar ciertas máquinas o aparatos para que empiecen a funcionar en el momento y en la forma deseados o elaborar programas para su empleo en computadoras.
En la actualidad, la noción de programación se encuentra muy asociada a la creación de aplicaciones de informática y videojuegos. En este sentido, es el proceso por el cual una persona desarrolla un programa, valiéndose de una herramienta que le permita escribir el código (el cual puede estar en uno o varios lenguajes. Como C++, Java y Python, entre otros)
En otras palabras programar es manipular datos. La programcaion es un proceso en el cual procesamos datos y obtener un resultado con los datos ya procesados.
Para qué sirve?

Por medio de la programación se diseña, codifica, limpia y protege el código fuente de los programas de los ordenadores.
A través de ella, también se diseña y crean aplicaciones o software. Como resultado se materializa un proyecto donde se utiliza un lenguaje de programación”.
Lenguajes de programación mas usados en la actualidad

1. Java
Reconocido por su legibilidad y simplicidad. Desde 2001 se mantiene en las primeras posiciones.

2. C
Es uno de los más utilizados en el mundo. Si bien es ejecutado en la mayoría de los sistemas operativos, es de propósito general, con lo cual es muy flexible.

3. Python
Un lenguaje de programación multiplataforma y multiparadigma.

4. C++
Conocido por el nombre “C Plus Plus”, se orienta a objetos surge como una continuación y ampliación del C.

5. C#
También llamado “C Sharp”, está orientado a objetos, C Sharp es una evolución del C y C++ que se destaca por su sencillez y modernidad.

6. JavaScript
No debemos confundirlo con Java. Son lenguajes distintos. Javascript es un lenguaje de programación que puede ser utilizado para crear programas que luego son acoplados a una página web o dentro de programas más grandes. Sirve para crear efectos y realizar acciones interactivas.
